Introduction

Computer software is a collection of programs which provides instructions telling the computer what to do. In simple terms, we can consider software as simply a set of instructions which direct the hardware to accomplish a task. Examples of software include Microsoft Office Word, Adobe Photoshop etc. The latter are able to tell the computer to do something. Adobe Photoshop for example will tell the computer how to change some properties of picture.

There are various types of software classified according their various functions. The two main classes of software are System software and application software. These are further classified into more specific groups depending on the type of task accomplished by each of the software.

System Software

When we talk of system software, we are referring to software which is designed to direct the functioning of the hardware and provide a base on which application software runs. This software has the ability to communicate directly with the physical components of the computer(hardware) passing to the latter tasks to be accomplished.

As relates to their interaction with application software, system software serve like a ground layer on which these applications run. This is because application software do not have the capability to communicate directly with the hardware. They therefore rely on system software in order to send and receive information from the hardware.

Software considered as system software include Operating systems and all utility software. Examples of operating systems include Windows NT, Windows 10, Linux, Unix, OS\2, MacOS, Chrome OS etc Utility software refers to software whose tasks are related to the analysis, configuration, optimization and/or maintenance of the computer system. Examples of utility programs include various types of antivirus software, backup software, compression utilities, debuggers, BIOS, boot loader etc

Application Software

Application software are software created to help the computer user accomplish more specific tasks. They are usually much easier to use and easy to learn. Application software can be classified into the following classes depending on the tasks they achieve:

  1. Word processors

    These are programs which can be used to create, edit, format, store and print out textual documents. They are the successors of the the orinal typewriter machine and make the creation of documents much easier. Examples include Microsoft Office word, LibreOffice Writer, Google Docs,Apple TextEdit etc.

  2. Spreadsheet software

    These are software capable of managing large amounts of data represented on spreadsheets commonly in numerical form. They are equipped with formulas and methods for performing various types of operations on the data such as plotting graphs,creating comments etc. Examples include Microsoft Office Excel, Google sheets, LibreOffice Calc, Lotus 1-2-3 etc

  3. Database Management Software (DBMS)

    A database is an organized collection of data for easy sorting, querrying, maintenance and updating. A DBMS is simply a software which has been created with the ability to manage such data collections. The offer means for computer users to conviniently manage large units of data. Examples of such software include MySQL,Oracle Database, Microsoft Office Access etc

  4. Desktop Publishing Software

    These are software which are used to create graphics for both desktop and publishing purposes. They can be used to create images, various types of cards, banners etc. Examples of such software include Adobe PageMaker, Adobe Photoshop, MS Publisher etc

  5. Internet browsers

    Browsers are one of the most common software in this era of information and communication technologies. They are simply computer programs that permit us to view web pages, read emails as well as access other online resources. The most commonly used browser nowadays on computers is Google Chrome browser. Other examples of browsers include Mozilla Firefox, Internet Explorer, UC Mini broser, Opera Browser etc

  6. Presentation Software

    These are software used to create multimedia presentations like slideshows, powerpoints etc. Examples include MS PowerPoint, Hyper Studio etc

  7. Others

    Apart form the above mentioned major classes of application software, there exist other classes like Sound editing software, picture editing software, Integrated Development Environments(IDEs) which are used for software production and a bunch many other types of software.

    It is worth noting that given the current state of our world, we can find software being used in every domain of life. It can therefore sometimes make sense to talk about classes of software as related to the domains of life where they are being used.
